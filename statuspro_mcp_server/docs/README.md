## Package Information

The MCP server is published as a separate package:

- **Package Name**: `statuspro-mcp-server`
- **PyPI**: https://pypi.org/project/statuspro-mcp-server/
- **Dependencies**: `statuspro-openapi-client`, `fastmcp`
- **Installation**: `pip install statuspro-mcp-server` or `uvx statuspro-mcp-server`
